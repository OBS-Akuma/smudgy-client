## How to Get Listed

Want to see your name here?

1. **Code contributions** - Submit a pull request with meaningful changes
2. **Subnames** - Add to the [subnames.json](https://github.com/OBS-Akuma/KirkaBadges/blob/main/Json/subnames.json) file
3. **Documentation** - Improve README, CONTRIBUTING, or other docs
4. **Community help** - Provide valuable feedback or help other users

After your contribution is merged, add yourself to this file in the same pull request!
